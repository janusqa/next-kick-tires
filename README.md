### Quickstart
- npx create-next-app@latest
  - Typescript:yes
  - ESLint: yes
  - Tailwind CSS: yes
  - src/ directory: no
  - App Router: yes
  - import alias: no
- npx shadcn-ui@latest init
  - Style: Default
  - base color: Slate
  - css variables for colors: yes
  
## Install shadcn indiviudal component
- npx shadcn-ui@latest add <component-name>
  - eg. npx shadcn-ui@latest add accordion
